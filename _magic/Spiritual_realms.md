# Spiritual Realms
Spiritual realms are vast conceptual forces that permeate the world. Each realm represents a fundamental aspect of existence — bog, darkness, fire, karst, molten sugar, and many others. People who develop connections to a realm can draw on it through [Channeling](/FATE_in_the_BAWG/magic/Channeling.html), [Adaptations](/FATE_in_the_BAWG/magic/Adaptations.html), [Attunement](/FATE_in_the_BAWG/magic/Attunement.html), or [Bargaining](/FATE_in_the_BAWG/magic/Bargaining.html).

Each realm has a set of **Truths** — the core facts that define what the realm is and does. These truths govern what channeling that realm can and cannot accomplish.

See the [Spiritual Realms](/FATE_in_the_BAWG/spiritual_realms/) collection for individual realm descriptions.
